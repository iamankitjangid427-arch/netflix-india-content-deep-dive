# 📌 Project Overview

This project provides an end-to-end analysis of **Netflix India** content using **Python, MySQL, and Power BI**.

The objective is to analyze Netflix India's content library, identify trends, generate business insights, and build an interactive Business Intelligence dashboard.

- Duration
- Genres

# 📌 Key Insights

- Total Indian Titles: **1046**
- Movies: **962**
- TV Shows: **84**
- TV-14 is the most common rating.
- International Movies dominate the catalog.
- Most Indian content was added between **2017–2020**.
